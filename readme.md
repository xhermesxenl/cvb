Crée un site web statique premium pour le :

# CVB
## Comité de Validation de la Bière

Le site doit être construit à partir de l'image présente dans le répertoire du projet (logo/écusson CVB).

────────────────────────────────────
CONTEXTE
────────────────────────────────────

Le CVB est une confrérie humoristique créée par un groupe d'amis.

Son univers est inspiré du jeu mobile de réalité augmentée Ingress de Niantic, et plus particulièrement de la faction Enlightened (ENL).

Les membres fondateurs proviennent de la communauté Enlightened de Montpellier (France).

IMPORTANT :

- Le site ne doit PAS être présenté comme un site officiel Ingress.
- Le site ne doit PAS être présenté comme un site officiel Niantic.
- Il s'agit d'une création communautaire humoristique.
- Il doit simplement évoquer l'esthétique, l'organisation et la culture des agents Enlightened.

L'univers du CVB repose sur l'idée absurde que la validation des bars est une mission officielle menée par une organisation semi-clandestine.

L'ambiance recherchée :

- cellule opérationnelle
- service de renseignement
- unité de terrain
- agence de certification
- confrérie secrète
- ordre de mission
- contrôle qualité

Le résultat doit ressembler davantage à :

"Une division spéciale Enlightened"

qu'à :

"Une brasserie artisanale"

ou

"Une marque de bière".
